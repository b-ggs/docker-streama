# Docker image for Streama

Docker image for [Streama](https://github.com/dularion/streama)

# TL;DR

```bash
docker run -d -v "/opt/dockerdata/streama/data:/data:rw" -p 8080:8080 --name "streama" bxggs/docker-streama
```
